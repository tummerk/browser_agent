# Тестирование Browser Service

Этот тестовый проект предназначен для проверки функциональности Browser Service в изоляции от других компонентов системы.

## Файлы

- `main.go` - Базовый тест, проверяющий основные функции
- `advanced_main.go` - Расширенный тест с анализом DOM и реальными действиями

## Запуск тестов

Для запуска базового теста:
```bash
go run main.go
```

Для запуска расширенного теста:
```bash
go run advanced_main.go
```

## Что тестируется

- Запуск браузера
- Навигация по страницам
- Сканирование DOM
- Поиск элементов в DOM
- Клик по элементам
- Ввод текста в поля
- Прокрутка страницы
- Комплексные сценарии (заполнение форм)
